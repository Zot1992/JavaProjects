Задание 1.
Написать класс ArrayList (на основе массива)
Необходимые методы:
-добавление(add)
-удаление(remove)
-получение(get)
-размер списка(size)

Необходимы также:
-конструктор по умолчанию
-конструктор копирования
-конструктор с начальным размером массива A_SIZE

Задание 2.
Написать программу, которая с помощью стека печатает дерево директорий(из директории запуска программы)

Задание 3.
Написать класс для работы с векторами(геометрия).
Необходимые методы:
-сумма с другим вектором
-разница с другим вектором
-длина вектора
-скалярное произведение
-нормализация
-получение угла с осью OX

Необходимы также:
-конструктор по умолчанию
-конструктор копирования

Задание 4.
Написать класс, реализующий бинарное дерево целых чисел.
Необходимые методы:
void add(int value)
boolean contains(int value)
void printAll()

Задание 5.
Написать графическую программу, выполняющую:
1) рисующую красный прямоугольник на черном экране
2) рисующую белый круг на черном экране
3) рисующую красную пятиконечную звезду
4) анимирующую движение звезды по оси OX
5) переделать задание 4 с помощью класса. Создать класс Star, позволяющий манипулировать звездой на холсте отрисовки.
6) отрисовку сетки с различным(рандомным) цветом клеток
